Blockchain for dapps without the downsides of current blockchains

Ethereum

  • Proof of Work (transitioning to Proof of Stake)
  • Transaction fees
  • New blocks ~15s
  • ~15 transactions/s
  • Ethereum addresses
  • Solidity as smart contract language
  • Gas is used to pay for smart contracts

EOS

  • Delegated Proof of Stake (DPos)
  • No transaction fees
  • New blocks ~0.5s
  • 3000+ transactions/s
  • Users are free to choose the account name
  • c++/webassembly as smart contract language
  • Coins are staked for the duration of a smart contract

Delegated Proof of Stake

All owners of EOS tokens can stake tokens as a vote for a block producer
The 21 most voted candidates are the 21 block producers
The 21 block producer create the new blocks; each block producer creates 6 blocks
Blocks need to be signed by at least 15 blocks
Users 'pay' for using EOS by a yearly 5% inflation
The amout of memory, storage, computing power and network traffic a user can use is based on the amount of EOS tokens

Governance

A Constitution consists of all non-programmeable rules; the hash of the constitution is used in all transactions
Changes in the constitution and source code need a 15/21 approval of the blockproducers for 30 consecutive days
Block producers can freeze accounts
